USER:
1. User can register themselves
2. User can deregister themselves
3. User can update their profile details
    - name
    - payment options
    - etc.

GROUP:
1. User can create groups
2. User can add other users to groups
3. User can remove users from the group

EXPENSE:
1. Group member can create new expense with sub-members of a group
2. Expense creator can add partipants to an expense
2. Expense creator can set split strategy
    - Equal split strategy
